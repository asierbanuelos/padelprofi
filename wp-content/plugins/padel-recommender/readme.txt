=== Padel Schläger Berater Pro ===
Version: 1.0.0
Requires at least: 5.8
Requires PHP: 7.4
Requires: WooCommerce

== Installation ==
1. ZIP hochladen → Plugins → Installieren → Aktivieren
2. Admin: 🏓 Schläger Berater → Marken konfigurieren
3. Zuordnungen: Spielerprofil wählen → pro Marke einen Schläger zuordnen
4. Shortcode [padel_recommender] auf gewünschter Seite einfügen

== Benutzung Admin ==
- Unter "Marken" alle Marken hinzufügen (Babolat, Adidas, NOX, etc.)
- Unter "Zuordnungen" für jede der 27 Kombinationen 
  (3 Spieler × 3 Niveaus × 3 Stile) einen Schläger pro Marke wählen
- Produkte kommen automatisch aus der WooCommerce-Kategorie "padelschlaeger"
- Mit Strg+S schnell speichern

== Frontend ==
- 3 Fragen: Spieler / Niveau / Spielstil  
- Zeigt eine Empfehlung pro Marke mit Bild, Preis und Link
- Vollständig auf Deutsch
